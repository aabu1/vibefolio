# Claude Code Portfolio Prompt (Impeccable)

Build a modern, minimal portfolio webpage using React + Tailwind.

Use a premium, clean aesthetic (similar to Linear, Vercel, or Stripe).
Prioritize typography, spacing, and subtle motion. This should feel like
a high-end product designer portfolio.

IMPORTANT: - Use the "Impeccable" skill to refine spacing, hierarchy,
and visual polish. - The design should feel intentional, not
template-y. - Avoid heavy colors --- mostly black, white, soft grays,
with one subtle accent.
